# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-12-27

### Added
- Initial release of JQL (JSON Query Language)
- Core `@jql/json-sql-explorer` library
- Monaco Editor integration with SQL auto-completion
- AlaSQL query execution engine
- Schema inference from JSON data
- Nested JSON flattening support
- Next.js App Router demo application
- TypeScript support with full type definitions
- Comprehensive test suite with Vitest
- SSR-safe implementation for Next.js

### Features
- SQL querying of JSON data in the browser
- Auto-completion for columns and SQL keywords
- Support for SELECT, WHERE, GROUP BY, ORDER BY, LIMIT
- Aggregate functions (COUNT, SUM, AVG, MIN, MAX)
- Light and dark themes
- Configurable result limits
- Error handling and validation
- Read-only mode option

### Developer Experience
- Monorepo structure with pnpm workspaces
- ESM and CJS build outputs
- Full TypeScript support
- Unit tests for core utilities
- ESLint and Prettier configuration
